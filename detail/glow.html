Glow

Glow (Global Weather) is an application that provides weather data for any location on Earth.

Features
  • Current weather forecast
  • Hourly weather forecast
  • Daily weather forecast
  • Search weather forecast by city
  • Set default city & temperature
  • Support dark mode
Technology used
  • Vue
  • Tailwind
  • Progressive Web App (PWA)
  • Indexed DB
  • Local Storage
